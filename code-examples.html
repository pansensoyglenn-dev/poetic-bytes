💡 Tips & Notes
⌨️

Keyboard Shortcut

Press Ctrl+Enter (or Cmd+Enter on Mac) anywhere in the editor to run your code instantly without reaching for the button.

📦

Standard Library Available

The full Python standard library is available: math, random, collections, itertools, functools, datetime, re, json, csv, io, and more.

🐍

Tab Indentation

Press Tab inside the editor to insert 4 spaces. Python requires consistent indentation — the runner honours your existing indent style.

🔁

State Persists Between Runs

Variables defined in one run are available in the next run within the same session. Clear Output resets the display, but not the Python namespace. Reload the page to reset everything.

Browser Sandbox Limitations

This runner uses Pyodide (Python compiled to WebAssembly). A few things aren't available in the browser sandbox: network requests (requests, urllib), file system access, subprocess, multiprocessing, and threading. Most standard library modules and pure-Python packages work fine. Scientific packages (numpy, pandas) can be loaded with import micropip; await micropip.install("numpy") but require an async cell.
