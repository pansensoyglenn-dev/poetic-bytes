INTRODUCTION

Strings are one of the most fundamental data types in Python. They represent sequences of Unicode characters and are immutable, meaning once created, their contents cannot be changed. This tutorial covers everything from basic operations to advanced techniques used in production environments.

FREQUENTLY ASKED QUESTIONS

WHY ARE PYTHON STRINGS IMMUTABLE?

Immutability enables strings to be dictionary keys, thread-safe, and allows memory optimizations like interning.

== VS IS FOR STRINGS?

== compares values, is compares identity (same object). Always use == for value equality.

WHEN SHOULD I USE RAW STRINGS?

Use raw strings (r"...") for regex patterns, file paths on Windows, and any string where backslashes should be literal characters. Example: r"\d+" for regex instead of "\\d+".

HOW DO I HANDLE UNICODE PROPERLY?

Always work with Unicode strings (str type) internally. Decode bytes to str at input boundaries and encode str to bytes at output boundaries. Use UTF-8 encoding unless you have a specific reason not to.

WHAT'S THE DIFFERENCE BETWEEN ENCODE() AND DECODE()?

encode() converts a str (Unicode) to bytes: "hello".encode('utf-8'). decode() converts bytes to str: b"hello".decode('utf-8'). Remember: str â†’ encode â†’ bytes â†’ decode â†’ str.

HOW CAN I CHECK IF A STRING CONTAINS ONLY CERTAIN CHARACTERS?

Use set operations or all() with a generator: all(c in 'abc123' for c in mystring). For performance, compile a regex pattern if checking repeatedly.

WHAT'S THE FASTEST WAY TO REVERSE A STRING?

Use slicing: reversed_str = mystring[::-1]. It's the most Pythonic and fastest approach. For very large strings, consider using reversed() iterator to save memory.

HOW DO I REMOVE ACCENTS FROM UNICODE STRINGS?

Use Unicode normalization: import unicodedata; unaccented = unicodedata.normalize('NFD', text).encode('ascii', 'ignore').decode(). This converts "cafÃ©" to "cafe".

SHOULD I USE f-STRINGS, format(), OR % FORMATTING?

Use f-strings for new code (fastest, most readable). Use format() if you need Python 3.5 compatibility. Avoid % formatting (legacy style). Exception: Use Template strings for untrusted user input.

HOW DO I SPLIT A STRING ON MULTIPLE DELIMITERS?

Use regex: import re; parts = re.split(r'[;,\s]+', text). This splits on semicolons, commas, or whitespace.
