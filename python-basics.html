🐍 print() – Your First Built-in Function

print() outputs text or values to the console. It’s the first function you’ll use in almost every Python program.

print("Hello, world!")
print("Python is fun!")
print(42)
$ python first.py
Hello, world!
Python is fun!
42

Step‑by‑step explanation:

  • print("Hello, world!") – calls the print() function with a string literal "Hello, world!". The function outputs that text to the console.
  • print("Python is fun!") – similarly, it prints the string "Python is fun!".
  • print(42) – print() can also output numbers, booleans, and other data types. Here it prints the integer 42.
  • Each print() call automatically adds a newline at the end, so the output appears on separate lines.
💡 You can pass any number of arguments to print() – they will be separated by a space by default. Try print("The answer is", 42).

🏷️ Variables & Assignment

Variables store data. In Python, you don’t declare a type – you just assign a value and Python figures it out.

📦 Creating Variables

Use the = operator to assign a value to a name. Variable names are case-sensitive and should be descriptive.

name = "Glenn"
age = 30
score = 98.5
is_active = True

print(name, age, score, is_active)
Glenn 30 98.5 True

Step‑by‑step explanation:

  • name = "Glenn" – creates a variable named name and assigns it the string "Glenn".
  • age = 30 – creates age and assigns the integer 30.
  • score = 98.5 – creates score and assigns the floating-point number 98.5.
  • is_active = True – creates is_active and assigns the boolean value True.
  • print(name, age, score, is_active) – passes all four variables to print(). The function prints them separated by spaces: Glenn 30 98.5 True.
⚠️ Python is dynamically typed – the same variable can hold different types at different times, but that can lead to bugs. Use type hints (we'll cover them later).

📝 Naming Conventions

  • ✓ Use lowercase letters, digits, and underscores (e.g., user_age)
  • ✓ Start with a letter or underscore (not a digit)
  • ✓ Avoid reserved keywords (if, for, while, etc.)
  • ✓ Use descriptive names – total_price is better than tp

📊 Python Data Types

Python has several built-in data types. The most common are integers, floats, strings, and booleans.

🔢

Integer (int)

Whole numbers, positive or negative, without decimal point.

age = 25
temperature = -5
population = 8_000_000_000  # underscores for readability
print(type(age))

Step‑by‑step: age = 25 assigns an integer; temperature = -5 negative integer; population = 8_000_000_000 uses underscores for readability (Python ignores them). print(type(age)) outputs <class 'int'>.

📏

Float (float)

Numbers with a decimal point or in scientific notation.

pi = 3.14159
height = 1.78
speed_of_light = 3e8  # 3 × 10⁸
print(type(pi))

Step‑by‑step: pi = 3.14159 assigns a float; height = 1.78 another float; speed_of_light = 3e8 uses scientific notation (3×10⁸). print(type(pi)) outputs <class 'float'>.

📝

String (str)

Text – enclosed in single, double, or triple quotes.

name = "Python"
message = 'Hello, world!'
multiline = """This is a
multi-line string."""
print(type(name))

Step‑by‑step: name = "Python" uses double quotes; message = 'Hello, world!' uses single quotes; multiline = """...""" triple quotes for multi‑line. print(type(name)) outputs <class 'str'>.

✅

Boolean (bool)

Represents truth values – either True or False.

is_python_fun = True
is_raining = False
print(type(is_python_fun))

Step‑by‑step: is_python_fun = True assigns boolean True; is_raining = False assigns False. print(type(is_python_fun)) outputs <class 'bool'>.

Check the type with type()

Use type(variable) to see the data type of any value. It’s great for debugging and learning.

⌨️ Getting User Input with input()

Interactive programs need to ask the user for information. input() reads a line from the console as a string.

📥 Basic Input

input() takes an optional prompt and always returns a string.

# Note: input() does not work in the browser.
# Copy this code to run it locally.
name = input("What is your name? ")
print(f"Hello, {name}!")
$ python input.py
What is your name? Glenn
Hello, Glenn!

Step‑by‑step explanation:

  • input("What is your name? ") – displays the prompt and waits for the user to type something and press Enter.
  • The entered text is returned as a string and assigned to name.
  • print(f"Hello, {name}!") – uses an f‑string to insert the value of name into the output string, printing a personalized greeting.
⚠️ The value returned is always a string. If you need a number, you must convert it – see the Type Conversion section below.
⚠️ input() does not work in the browser. Use the Copy button to run locally.

💬 Comments & Code Readability

Comments are for humans – they explain what your code does and why. Python ignores them.

# Single-line comments

Start a comment with # – everything after is ignored.

# Calculate the area of a rectangle
length = 10
width = 5
area = length * width
print(area)

Step‑by‑step explanation:

  • # Calculate the area of a rectangle – a comment, ignored by Python, explaining the purpose of the following code.
  • length = 10 – assigns integer 10 to variable length.
  • width = 5 – assigns integer 5 to width.
  • area = length * width – computes the product and stores it in area.
  • print(area) – outputs the calculated area, which is 50.

""" Multi-line comments (docstrings) """

Triple-quoted strings that are not assigned to anything can serve as documentation.

"""This is a multi-line comment.
It can span several lines.
Used often at the top of a file or function."""
print("Hello")

Step‑by‑step explanation:

  • """This is a multi-line comment... – a triple‑quoted string that is not assigned to any variable; Python treats it as a docstring (or just a string literal) and ignores it if not used.
  • print("Hello") – prints Hello to the console.
💡 Good code is self-documenting – clear variable and function names reduce the need for comments. Use comments to explain the why, not the what.

➕ Basic Arithmetic Operations

Python supports all the usual arithmetic operators.

a = 10
b = 3

print("Addition:", a + b)
print("Subtraction:", a - b)
print("Multiplication:", a * b)
print("Division (float):", a / b)
print("Floor division:", a // b)
print("Modulo (remainder):", a % b)
print("Exponentiation:", a ** b)
Addition: 13
Subtraction: 7
Multiplication: 30
Division (float): 3.3333333333333335
Floor division: 3
Modulo (remainder): 1
Exponentiation: 1000

Step‑by‑step explanation:

  • a = 10; b = 3 – assign integers.
  • a + b → 13 (addition).
  • a - b → 7 (subtraction).
  • a * b → 30 (multiplication).
  • a / b → 3.333… (true division, always returns a float).
  • a // b → 3 (floor division, discards remainder).
  • a % b → 1 (modulo, remainder of division).
  • a ** b → 1000 (exponentiation, 10³).
✨ Python 3.12+ allows underscores in numeric literals for readability, e.g., 1_000_000.

🔤 String Manipulation & f-strings

Strings are sequences of characters. Python makes them easy to combine, slice, and format.

➕ Concatenation

Use + to join strings, but be careful – it only works with strings.

first = "Python"
last = "Rocks"
full = first + " " + last
print(full)

Step‑by‑step explanation:

  • first = "Python" – string variable.
  • last = "Rocks" – another string.
  • full = first + " " + last – concatenates three strings: "Python", a space, and "Rocks" → "Python Rocks".
  • print(full) – prints Python Rocks.

📝 f-strings – the modern way

Prefix a string with f and put variables in {} – Python inserts their values.

name = "Alice"
score = 95.5
print(f"Hello, {name}! Your score is {score:.1f}%")
Hello, Alice! Your score is 95.5%

Step‑by‑step explanation:

  • name = "Alice" – string.
  • score = 95.5 – float.
  • print(f"Hello, {name}! Your score is {score:.1f}%") – the f prefix makes it an f‑string. Inside {} we place variable names; {score:.1f} formats the float to one decimal place. The result: Hello, Alice! Your score is 95.5%.

🔪 Slicing & Methods

Strings support slicing and many useful methods.

text = "Python Programming"
print(text[0:6])
print(text.upper())
print(text.replace("Programming", "Basics"))
Python
PYTHON PROGRAMMING
Python Basics

Step‑by‑step explanation:

  • text = "Python Programming" – string.
  • text[0:6] – slice from index 0 (inclusive) to 6 (exclusive) → "Python".
  • text.upper() – method that returns the string in uppercase → "PYTHON PROGRAMMING".
  • text.replace("Programming", "Basics") – replaces all occurrences of "Programming" with "Basics" → "Python Basics".

🔄 Type Conversion (Casting)

Sometimes you need to convert one type to another – e.g., a string to an integer.

# Note: input() does not work in the browser.
# Copy this code to run it locally.
age_input = input("Enter your age: ")
age = int(age_input)
print(f"Next year you'll be {age + 1}")

pi = 3.14159
pi_int = int(pi)
print(pi_int)

number = 42
number_str = str(number)
print("The answer is " + number_str)

Step‑by‑step explanation:

  • age_input = input("Enter your age: ") – reads a string.
  • age = int(age_input) – converts that string to an integer so we can do arithmetic.
  • print(f"Next year you'll be {age + 1}") – adds 1 to the integer and prints.
  • pi = 3.14159; pi_int = int(pi) – truncates the float to integer (3).
  • print(pi_int) → 3.
  • number = 42; number_str = str(number) – converts integer to string.
  • print("The answer is " + number_str) – concatenates two strings.

Common conversion functions

int(), float(), str(), bool() – these are your tools for converting between types.
