Getting Started with Python

Python is a high-level, interpreted programming language known for its simplicity and readability. It's perfect for beginners and used widely in web development, data science, AI, and more.

Why Python?

Python has a clean syntax that looks like plain English, making it easier to learn than many other programming languages. It also has a huge community and extensive libraries.

Your First Python Program

# This is a comment in Python
print("Hello, World!")  # This prints text to the console

# Simple arithmetic
result = 5 + 3
print("5 + 3 =", result)

Python Syntax Basics

Python uses indentation to define code blocks instead of curly braces. This makes Python code very readable.

Indentation Matters

# Correct indentation
if 5 > 2:
    print("Five is greater than two!")
    print("This is also inside the if block")

print("This is always printed")

Comments

Use # for single-line comments. Comments are ignored by Python.

Strings

Text can be in single 'quotes' or double "quotes".

Variables

No need to declare variable types. Just assign values: x = 5

Variables and Data Types

In Python, you don't need to declare variable types. Python automatically determines the type based on the value assigned.

Basic Data Types

# Integer
age = 25

# Float (decimal number)
price = 19.99

# String (text)
name = "Alice"

# Boolean (True or False)
is_student = True

# List
fruits = ["apple", "banana", "cherry"]

# Dictionary
person = {"name": "Bob", "age": 30}

Control Flow

Control flow statements let you execute code conditionally or repeatedly.

If-Else Statements

# If statement
temperature = 25

if temperature > 30:
    print("It's hot outside!")
elif temperature > 20:
    print("It's warm outside.")
else:
    print("It's cool outside.")

Functions

Functions are reusable blocks of code that perform a specific task. They help organize code and avoid repetition.

Defining and Calling Functions

# Define a function
def greet(name):
    """This function greets a person"""
    return f"Hello, {name}!"

# Call the function
message = greet("Alice")
print(message)  # Output: Hello, Alice!
