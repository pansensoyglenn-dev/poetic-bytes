📜 History of Python

From a Hobby Project to the World's Most Popular Language

Python was created in the late 1980s by Guido van Rossum as a hobby project to keep him busy during the Christmas holidays. He named it after the British comedy troupe Monty Python — not the snake! What started as a personal project has grown into the most widely used programming language on the planet, powering everything from YouTube to NASA's space missions.

1989 — The Beginning

Guido van Rossum starts working on Python as a "hobby project" during Christmas. He wants to create a language that's easy to read and fun to use.

1991 — First Release

Python 0.9.0 is released. It already includes classes, functions, exception handling, and the core data types — lists, strings, and dictionaries.

1994 — Python 1.0

The first major milestone. Python 1.0 introduces functional programming tools like lambda, map, and filter.

2000 — Python 2.0

Python 2.0 introduces list comprehensions, garbage collection, and Unicode support. It becomes the standard for many years.

2008 — Python 3.0

A major overhaul to fix language inconsistencies. Python 3 is not backward‑compatible with Python 2 — a deliberate decision to improve the language.

2020 — Python 2 Retired

After 20 years, Python 2 is officially sunset. The community fully transitions to Python 3, which continues to evolve with new features every year.

Today — Python Everywhere

Python is the #1 language for data science, AI, web development, and automation. It's taught in schools, used by Fortune 500 companies, and loved by millions.

🔧 Setting Up Your Python Environment

Before you write your first line of code, you need Python installed on your machine. Here's how to get started.

🔧 1.1 Installing Python

Python is free and available for Windows, macOS, and Linux. Follow these steps to get Python on your computer.

Step-by-step guide:

  • Step 1: Go to python.org and download the latest Python 3 version for your operating system.
  • Step 2: Run the installer. On Windows, make sure to check "Add Python to PATH" before clicking Install.
  • Step 3: Verify the installation by opening a terminal (Command Prompt on Windows, Terminal on macOS/Linux) and typing python --version.
  • Step 4: You should see something like Python 3.12.x — this means Python is installed and ready.
  • Step 5: Try running Python interactively by typing python in the terminal. You'll see a >>> prompt where you can type Python code directly.
terminal / python 
# Check Python version
                            # Open your terminal and run:
                            # python --version

                            # Start Python interactively:
                            # python

                            # Then type your first Python code:
                            print("Hello, World!")

                            # Exit the interactive shell:
                            # exit()

📦 Variables and Data Types

Variables are named containers for data. Learn how to store and work with different types of data in Python.

📦 2.1 Variables and Basic Types

In Python, you don't need to declare the type of a variable — Python figures it out automatically. This is called dynamic typing.

Step-by-step explanation:

  • Step 1: Assign a value to a variable using the = operator.
  • Step 2: Python infers the type from the value: int for whole numbers, float for decimal numbers, str for text, bool for True/False.
  • Step 3: Use the type() function to check the type of a variable.
  • Step 4: Variables can be reassigned to different types — Python is flexible.
  • Step 5: Variable names are case‑sensitive and should be descriptive.
python 
# Integer (whole number)
                            age = 25
                            print("Age:", age)
                            print("Type of age:", type(age))

                            # Float (decimal number)
                            price = 19.99
                            print("Price:", price)
                            print("Type of price:", type(price))

                            # String (text)
                            name = "Alice"
                            print("Name:", name)
                            print("Type of name:", type(name))

                            # Boolean (True/False)
                            is_student = True
                            print("Is student:", is_student)
                            print("Type of is_student:", type(is_student))

                            # Reassigning a variable
                            age = "twenty-five"  # Now age is a string!
                            print("Age now:", age)
                            print("Type now:", type(age))
Python's dynamic typing makes code fast to write and easy to read. Unlike languages like C or Java, you don't need to declare types — Python handles it for you.

🔤 Strings — Working with Text

Strings are sequences of characters. Python provides powerful tools for manipulating and formatting text.

🔤 3.1 String Operations and Methods

Strings can be combined, sliced, and transformed using built‑in methods.

Step-by-step explanation:

  • Step 1: Create strings with single quotes ' ' or double quotes " ".
  • Step 2: Concatenate strings with the + operator.
  • Step 3: Access individual characters using indexing (0‑based).
  • Step 4: Use methods like .upper(), .lower(), .strip() to transform strings.
  • Step 5: Use len() to get the length of a string.
python 
# Creating strings
                            greeting = "Hello"
                            name = "World"
                            message = greeting + " " + name + "!"
                            print("Message:", message)

                            # String indexing
                            word = "Python"
                            print("First character:", word[0])
                            print("Last character:", word[-1])
                            print("Slice (0:3):", word[0:3])

                            # String methods
                            text = "  Hello, World!  "
                            print("Original:", text)
                            print("Upper:", text.upper())
                            print("Lower:", text.lower())
                            print("Strip:", text.strip())
                            print("Title:", text.strip().title())

                            # Replace
                            print("Replace:", text.replace("World", "Python"))

                            # Length
                            print("Length:", len(text))
Strings are immutable — you can't change a character in place. Methods like .upper() return a new string rather than modifying the original.

🔢 Numbers and Arithmetic

Python supports integers, floating‑point numbers, and complex numbers. Learn to perform arithmetic operations.

🔢 4.1 Arithmetic Operations

Python has all the standard arithmetic operators: addition, subtraction, multiplication, division, and more.

Step-by-step explanation:

  • Step 1: Use +, -, *, / for basic arithmetic.
  • Step 2: Use // for floor division (integer result).
  • Step 3: Use % for modulo (remainder).
  • Step 4: Use ** for exponentiation.
  • Step 5: Python follows standard operator precedence (PEMDAS).
python 
# Basic arithmetic
                            a = 10
                            b = 3

                            print("a + b =", a + b)
                            print("a - b =", a - b)
                            print("a * b =", a * b)
                            print("a / b =", a / b)  # Division always returns a float
                            print("a // b =", a // b)  # Floor division
                            print("a % b =", a % b)    # Modulo (remainder)
                            print("a ** b =", a ** b)  # Exponentiation

                            # Operator precedence
                            result = 2 + 3 * 4 ** 2
                            print("2 + 3 * 4 ** 2 =", result)

                            # Using parentheses for clarity
                            result2 = (2 + 3) * (4 ** 2)
                            print("(2 + 3) * (4 ** 2) =", result2)

                            # Type conversion
                            x = 5
                            y = 2.5
                            print("x + y =", x + y)  # int + float = float
                            print("int(2.9) =", int(2.9))  # Truncates decimal
                            print("float(5) =", float(5))
Division (/) always returns a float. If you need an integer result, use floor division (//).

📋 Lists — Ordered Collections

Lists are ordered, mutable collections that can hold any type of data. They're one of the most useful data structures in Python.

📋 5.1 Working with Lists

Lists can contain mixed types, support indexing and slicing, and have many useful methods.

Step-by-step explanation:

  • Step 1: Create a list using square brackets [].
  • Step 2: Access elements by index (0‑based).
  • Step 3: Modify elements by assigning to an index.
  • Step 4: Use methods: .append(), .insert(), .remove(), .pop().
  • Step 5: Use len() to get the number of elements.
python 
# Creating lists
                            fruits = ["apple", "banana", "orange", "grape"]
                            print("Fruits:", fruits)

                            # Accessing elements
                            print("First fruit:", fruits[0])
                            print("Last fruit:", fruits[-1])
                            print("First two:", fruits[0:2])

                            # Modifying elements
                            fruits[1] = "blueberry"
                            print("After modification:", fruits)

                            # Adding elements
                            fruits.append("kiwi")
                            print("After append:", fruits)
                            fruits.insert(1, "mango")
                            print("After insert:", fruits)

                            # Removing elements
                            fruits.remove("orange")
                            print("After remove:", fruits)
                            last = fruits.pop()
                            print("Popped:", last)
                            print("After pop:", fruits)

                            # Other useful methods
                            numbers = [3, 1, 4, 1, 5, 9, 2]
                            numbers.sort()
                            print("Sorted:", numbers)
                            numbers.reverse()
                            print("Reversed:", numbers)
                            print("Count of 1:", numbers.count(1))
                            print("Index of 5:", numbers.index(5))
Lists are mutable — you can change, add, and remove elements after creation.

📌 Tuples — Immutable Sequences

Tuples are like lists but immutable — once created, they cannot be changed. Use tuples for data that shouldn't be modified.

📌 6.1 Working with Tuples

Tuples are defined with parentheses () and support indexing and unpacking.

Step-by-step explanation:

  • Step 1: Create a tuple using parentheses ().
  • Step 2: Access elements by index (same as lists).
  • Step 3: Tuple unpacking: assign tuple values to multiple variables.
  • Step 4: Tuples are immutable — you cannot modify them.
  • Step 5: Use tuples to group related data together.
python 
# Creating tuples
                            coordinates = (10, 20)
                            rgb = (255, 128, 0)
                            person = ("Alice", 25, "Engineer")
                            print("Coordinates:", coordinates)
                            print("RGB:", rgb)
                            print("Person:", person)

                            # Accessing elements
                            print("X:", coordinates[0])
                            print("Y:", coordinates[1])
                            print("First color:", rgb[0])

                            # Tuple unpacking
                            x, y = coordinates
                            print("Unpacked x:", x)
                            print("Unpacked y:", y)

                            name, age, job = person
                            print("Name:", name)
                            print("Age:", age)
                            print("Job:", job)

                            # Tuples are immutable
                            # rgb[0] = 100  # This will raise an error!

                            # Single-element tuple (note the comma)
                            single = (42,)
                            print("Single element tuple:", single)  # Not an int!

                            # Tuple as a return value from a function
                            def get_stats(numbers):
                            return min(numbers), max(numbers), sum(numbers)/len(numbers)

                            min_val, max_val, avg = get_stats([5, 10, 15, 20])
                            print("Min:", min_val, "Max:", max_val, "Average:", avg)
Use tuples when you want to guarantee that data won't be accidentally modified. They're also more memory‑efficient than lists.

📖 Dictionaries — Key‑Value Pairs

Dictionaries store data as key‑value pairs. They're like real‑world dictionaries — you look up a word (key) to find its definition (value).

📖 7.1 Working with Dictionaries

Dictionaries are mutable, unordered collections of key‑value pairs. Keys must be unique and immutable (strings, numbers, tuples).

Step-by-step explanation:

  • Step 1: Create a dictionary with curly braces {} and colons :.
  • Step 2: Access values using square bracket notation dict['key'].
  • Step 3: Add or modify key‑value pairs by assignment.
  • Step 4: Use .get() to safely access values (returns None if key doesn't exist).
  • Step 5: Iterate over keys, values, or items using loops.
python 
# Creating a dictionary
                            person = {
                            "name": "Alice",
                            "age": 30,
                            "city": "New York",
                            "job": "Engineer"
                            }
                            print("Person:", person)

                            # Accessing values
                            print("Name:", person["name"])
                            print("Age:", person["age"])

                            # Using .get() to avoid KeyError
                            print("Country:", person.get("country", "Unknown"))

                            # Adding and modifying
                            person["country"] = "USA"
                            person["age"] = 31
                            print("Updated:", person)

                            # Getting all keys, values, items
                            print("Keys:", person.keys())
                            print("Values:", person.values())
                            print("Items:", person.items())

                            # Iterating over a dictionary
                            print("\nIterating over items:")
                            for key, value in person.items():
                            print(f"{key}: {value}")

                            # Dictionary comprehension
                            squares = {x: x**2 for x in range(5)}
                            print("Squares:", squares)

                            # Nested dictionaries
                            users = {
                            "alice": {"age": 30, "city": "NYC"},
                            "bob": {"age": 25, "city": "LA"}
                            }
                            print("Alice's city:", users["alice"]["city"])
Dictionary keys must be immutable (strings, numbers, tuples). Lists and other dictionaries cannot be used as keys.

🧩 Sets — Unordered Unique Collections

Sets store unique elements in no particular order. They're useful for membership testing and removing duplicates.

🧩 8.1 Working with Sets

Sets are defined with curly braces {} and support mathematical set operations.

Step-by-step explanation:

  • Step 1: Create a set with curly braces {} or set().
  • Step 2: Sets automatically remove duplicates.
  • Step 3: Use .add() and .remove() to modify sets.
  • Step 4: Perform set operations: union, intersection, difference.
  • Step 5: Use sets for fast membership testing.
python 
# Creating sets
                            fruits = {"apple", "banana", "orange", "apple", "banana"}
                            print("Fruits (duplicates removed):", fruits)

                            # Creating a set from a list
                            numbers = [1, 2, 3, 2, 1, 4, 5]
                            unique = set(numbers)
                            print("Unique numbers:", unique)

                            # Adding and removing
                            fruits.add("grape")
                            print("After add:", fruits)
                            fruits.remove("banana")
                            print("After remove:", fruits)

                            # Set operations
                            set_a = {1, 2, 3, 4, 5}
                            set_b = {4, 5, 6, 7, 8}

                            print("Union:", set_a | set_b)
                            print("Intersection:", set_a & set_b)
                            print("Difference (A - B):", set_a - set_b)
                            print("Symmetric difference:", set_a ^ set_b)

                            # Membership testing
                            print("Is 3 in set_a?", 3 in set_a)
                            print("Is 10 in set_a?", 10 in set_a)

                            # Set comprehension
                            squares = {x**2 for x in range(6)}
                            print("Squares set:", squares)
Membership testing in sets is O(1) (constant time) — much faster than lists for large collections.

🔀 Conditionals — Making Decisions

Conditionals allow your code to make decisions based on conditions. Use if, elif, and else.

🔀 9.1 If, Elif, Else Statements

Python uses indentation (whitespace) to define code blocks. This makes the code clean and readable.

Step-by-step explanation:

  • Step 1: Use if followed by a condition and a colon :.
  • Step 2: Indent the code block that runs if the condition is True.
  • Step 3: Use elif (else if) for additional conditions.
  • Step 4: Use else for the fallback when all conditions are False.
  • Step 5: Conditions use comparison operators: ==, !=, >, <, >=, <=.
python 
# Basic if statement
                            age = 20
                            if age >= 18:
                            print("You are an adult")
                            else:
                            print("You are a minor")

                            # Using elif
                            score = 85
                            if score >= 90:
                            grade = "A"
                            elif score >= 80:
                            grade = "B"
                            elif score >= 70:
                            grade = "C"
                            elif score >= 60:
                            grade = "D"
                            else:
                            grade = "F"
                            print("Grade:", grade)

                            # Multiple conditions
                            temperature = 25
                            is_sunny = True
                            if temperature > 30:
                            print("It's hot")
                            elif temperature > 20 and is_sunny:
                            print("It's warm and sunny")
                            elif temperature > 20:
                            print("It's warm")
                            else:
                            print("It's cool")

                            # Nested conditionals
                            x = 10
                            if x > 0:
                            if x % 2 == 0:
                            print("x is positive and even")
                            else:
                            print("x is positive and odd")
                            else:
                            print("x is not positive")

                            # Ternary operator (conditional expression)
                            age = 16
                            status = "Adult" if age >= 18 else "Minor"
                            print("Status:", status)
Python uses indentation to define blocks — not curly braces or keywords like end. This makes Python code clean and forces consistent formatting.

🔄 Loops — Repeating Actions

Loops allow you to execute a block of code multiple times. Python has for loops and while loops.

🔄 10.1 For and While Loops

for loops iterate over sequences (lists, strings, dictionaries). while loops run while a condition is True.

Step-by-step explanation:

  • Step 1: Use for to iterate over a sequence.
  • Step 2: Use range() to generate numeric sequences.
  • Step 3: Use while when the number of iterations is not known in advance.
  • Step 4: Use break to exit a loop early.
  • Step 5: Use continue to skip the rest of the current iteration.
python 
# For loop over a list
                            fruits = ["apple", "banana", "orange"]
                            for fruit in fruits:
                            print("I like", fruit)

                            # For loop with range()
                            print("\nCounting with range:")
                            for i in range(5):
                            print(i, end=" ")
                            print()

                            # For loop with index
                            print("\nWith index:")
                            for i, fruit in enumerate(fruits):
                            print(f"{i}: {fruit}")

                            # While loop
                            print("\nWhile loop:")
                            count = 0
                            while count < 5:
                            print(count, end=" ")
                            count += 1
                            print()

                            # Break and continue
                            print("\nBreak and continue:")
                            for i in range(10):
                            if i == 3:
                            continue  # Skip 3
                            if i == 7:
                            break     # Stop at 7
                            print(i, end=" ")
                            print()

                            # Looping over a dictionary
                            person = {"name": "Alice", "age": 30, "city": "NYC"}
                            print("\nDictionary loop:")
                            for key, value in person.items():
                            print(f"{key}: {value}")

                            # Nested loops
                            print("\nNested loops:")
                            for i in range(3):
                            for j in range(2):
                            print(f"({i},{j})", end=" ")
                            print()
range(n) generates numbers from 0 to n-1. Use range(start, stop, step) for more control.

⚙️ Functions — Reusable Code Blocks

Functions allow you to encapsulate reusable code. They take inputs (parameters) and can return outputs.

⚙️ 11.1 Defining and Using Functions

Functions are defined with the def keyword. They can have parameters and return values.

Step-by-step explanation:

  • Step 1: Define a function with def function_name(parameters):.
  • Step 2: Indent the function body.
  • Step 3: Use return to send a value back to the caller.
  • Step 4: Call the function by name with arguments.
  • Step 5: Functions can have default parameter values.
python 
# Defining a simple function
                            def greet(name):
                            return f"Hello, {name}!"

                            print(greet("Alice"))
                            print(greet("Bob"))

                            # Function with multiple parameters
                            def add(a, b):
                            return a + b

                            result = add(5, 3)
                            print("5 + 3 =", result)

                            # Function with default parameters
                            def power(base, exponent=2):
                            return base ** exponent

                            print("3^2 =", power(3))
                            print("2^3 =", power(2, 3))

                            # Function with variable number of arguments
                            def sum_all(*numbers):
                            return sum(numbers)

                            print("Sum of 1, 2, 3:", sum_all(1, 2, 3))
                            print("Sum of 1, 2, 3, 4, 5:", sum_all(1, 2, 3, 4, 5))

                            # Function with keyword arguments
                            def describe_person(name, age, city="Unknown"):
                            return f"{name} is {age} years old and lives in {city}"

                            print(describe_person("Alice", 30, "NYC"))
                            print(describe_person("Bob", 25))

                            # Lambda functions (anonymous functions)
                            square = lambda x: x ** 2
                            print("Lambda square of 5:", square(5))

                            # Functions as arguments
                            def apply(func, value):
                            return func(value)

                            print("apply(square, 4):", apply(square, 4))
Functions help you write DRY (Don't Repeat Yourself) code. They make your programs more organized, testable, and reusable.

⚠️ Error Handling — Try / Except

Errors happen. Learn to handle them gracefully so your program doesn't crash unexpectedly.

⚠️ 12.1 Exception Handling

Use try and except to catch and handle exceptions (errors) in your code.

Step-by-step explanation:

  • Step 1: Wrap code that might raise an error in a try block.
  • Step 2: Catch specific exceptions with except.
  • Step 3: Use else for code that runs if no exception occurs.
  • Step 4: Use finally for code that runs regardless of whether an exception occurred.
  • Step 5: Raise your own exceptions with raise.
python 
# Basic try/except
                            try:
                            number = int(input("Enter a number: "))
                            print("You entered:", number)
                            except ValueError:
                            print("That's not a valid number!")

                            # Catching specific exceptions
                            try:
                            result = 10 / 0
                            except ZeroDivisionError:
                            print("Cannot divide by zero!")
                            except TypeError:
                            print("Invalid type for division!")
                            except Exception as e:
                            print("Something went wrong:", e)

                            # Try with else and finally
                            try:
                            file = open("example.txt", "r")
                            content = file.read()
                            except FileNotFoundError:
                            print("File not found!")
                            else:
                            print("File content length:", len(content))
                            finally:
                            try:
                            file.close()
                            except:
                            pass
                            print("Cleanup complete.")

                            # Raising exceptions
                            def validate_age(age):
                            if age < 0:
                            raise ValueError("Age cannot be negative")
                            elif age > 150:
                            raise ValueError("Age cannot be greater than 150")
                            return f"Valid age: {age}"

                            try:
                            print(validate_age(25))
                            print(validate_age(-5))
                            except ValueError as e:
                            print("Error:", e)
Always catch specific exceptions when possible. Catching Exception can hide bugs. Use finally for resource cleanup.

📁 File I/O — Reading and Writing Files

Learn to read from and write to files on your computer. This is essential for working with data and saving program results.

📁 14.1 Reading and Writing Files

Use the open() function to work with files. Always close files or use the with context manager.

Step-by-step explanation:

  • Step 1: Open a file with open(filename, mode).
  • Step 2: Modes: 'r' (read), 'w' (write), 'a' (append).
  • Step 3: Read content with .read(), .readline(), or .readlines().
  • Step 4: Write content with .write().
  • Step 5: Always close files with .close() or use with.
python 
# Writing to a file
                            with open("example.txt", "w") as file:
                            file.write("Hello, World!\n")
                            file.write("This is line 2.\n")
                            file.write("This is line 3.\n")
                            print("File written successfully.")

                            # Reading the entire file
                            with open("example.txt", "r") as file:
                            content = file.read()
                            print("File content:")
                            print(content)

                            # Reading line by line
                            print("Reading line by line:")
                            with open("example.txt", "r") as file:
                            for line in file:
                            print(f"Line: {line.strip()}")

                            # Reading all lines into a list
                            with open("example.txt", "r") as file:
                            lines = file.readlines()
                            print("Lines list:", lines)

                            # Appending to a file
                            with open("example.txt", "a") as file:
                            file.write("This line was appended.\n")
                            print("Appended successfully.")

                            # Writing a list of strings
                            fruits = ["apple\n", "banana\n", "orange\n"]
                            with open("fruits.txt", "w") as file:
                            file.writelines(fruits)

                            # Reading with error handling
                            try:
                            with open("nonexistent.txt", "r") as file:
                            content = file.read()
                            except FileNotFoundError:
                            print("File not found!")

                            # CSV-like file handling
                            data = [
                            ["Alice", "30", "Engineer"],
                            ["Bob", "25", "Designer"],
                            ["Charlie", "35", "Manager"]
                            ]
                            with open("people.csv", "w") as file:
                            for row in data:
                            file.write(",".join(row) + "\n")
                            print("CSV written.")
The with statement automatically closes the file when you're done — even if an error occurs. Always use it when working with files.

🏗️ Classes and Object‑Oriented Programming

Classes allow you to create your own data types with properties and methods. OOP helps you structure large programs.

🏗️ 15.1 Defining and Using Classes

Classes are blueprints for objects. They encapsulate data (attributes) and behavior (methods).

Step-by-step explanation:

  • Step 1: Define a class with class ClassName:.
  • Step 2: Use __init__ as the constructor to set up new objects.
  • Step 3: Use self to refer to the current instance.
  • Step 4: Define methods (functions inside classes) that operate on the object.
  • Step 5: Create objects (instances) from the class.
python 
# Defining a simple class
                            class Person:
                            def __init__(self, name, age):
                            self.name = name
                            self.age = age

                            def greet(self):
                            return f"Hello, my name is {self.name} and I am {self.age} years old."

                            def birthday(self):
                            self.age += 1
                            print(f"{self.name} is now {self.age} years old!")

                            # Creating objects
                            alice = Person("Alice", 30)
                            bob = Person("Bob", 25)

                            print(alice.greet())
                            print(bob.greet())

                            alice.birthday()
                            bob.birthday()

                            # Class variables
                            class Student:
                            school = "Python University"  # Class variable shared by all instances

                            def __init__(self, name, grade):
                            self.name = name
                            self.grade = grade

                            def display(self):
                            return f"{self.name} (Grade: {self.grade}) at {Student.school}"

                            s1 = Student("Alice", "A")
                            s2 = Student("Bob", "B")
                            print(s1.display())
                            print(s2.display())

                            # Inheritance
                            class Animal:
                            def __init__(self, name):
                            self.name = name

                            def speak(self):
                            return "Some sound"

                            class Dog(Animal):
                            def speak(self):
                            return "Woof!"

                            class Cat(Animal):
                            def speak(self):
                            return "Meow!"

                            dog = Dog("Rex")
                            cat = Cat("Whiskers")
                            print(f"{dog.name} says: {dog.speak()}")
                            print(f"{cat.name} says: {cat.speak()}")

                            # Property decorator (getter)
                            class Rectangle:
                            def __init__(self, width, height):
                            self._width = width
                            self._height = height

                            @property
                            def area(self):
                            return self._width * self._height

                            @property
                            def perimeter(self):
                            return 2 * (self._width + self._height)

                            rect = Rectangle(4, 5)
                            print(f"Area: {rect.area}")
                            print(f"Perimeter: {rect.perimeter}")
Object‑Oriented Programming (OOP) is a powerful paradigm. It helps you model real‑world entities and build complex, maintainable software.
