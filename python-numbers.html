💧 Floating-Point Numbers (float) — Real Numbers with Precision

Python floats are double-precision (64-bit) IEEE 754 values. They can represent numbers with fractional parts.

💧 2.1 Float Literals and Operations

  • Step 4: Declare a very large float: 1.23e8 equals 123000000.0.
  • Step 5: Perform addition, subtraction, multiplication, and division.
  • Step 6: Show the precision issue: 0.1 + 0.2 == 0.3 is False due to binary representation.

  • Step 8: Call the function to execute the code.
python 
def float_demo():
    pi = 3.14159
    small = 1.5e-3
    large = 1.23e8

    print(f"pi = {pi}")
    print(f"1.5e-3 = {small}")
    print(f"1.23e8 = {large}")

    a, b = 5.2, 2.3
    print(f"\n{a} + {b} = {a + b}")
    print(f"{a} - {b} = {a - b}")
    print(f"{a} * {b} = {a * b}")
    print(f"{a} / {b} = {a / b}")

    print("\n--- Precision issue ---")
    print(f"0.1 + 0.2 = {0.1 + 0.2}")
    print(f"0.1 + 0.2 == 0.3 is {0.1 + 0.2 == 0.3}")
    print(f"abs((0.1 + 0.2) - 0.3) < 1e-9 is {abs((0.1 + 0.2) - 0.3) < 1e-9}")

⚠️ Floating-point arithmetic is not exact for many decimal fractions. Use decimal.Decimal for financial calculations where exact decimal representation is required.

Calculate compound interest with Decimal for precise monetary values.

  • Step 2: Define compound_interest() with parameters: principal, rate, years, and compounding periods.
  • Step 3: Use Decimal for all monetary values to avoid floating-point errors.
  • Step 4: Calculate the compound interest formula: P * (1 + r/n)^(n*t).
  • Step 5: Format the result with 2 decimal places using quantize().
  • Step 6: In main(), call the function with sample values and print the result.
  • Step 7: Call the main function to execute the code.
python 
from decimal import Decimal, getcontext

def compound_interest(principal, rate, years, n=12):
    getcontext().prec = 28

    p = Decimal(str(principal))
    r = Decimal(str(rate))
    t = Decimal(str(years))
    n_dec = Decimal(str(n))

    amount = p * (1 + r / n_dec) ** (n_dec * t)
    return amount.quantize(Decimal("0.01"))

def main():
    principal = 1000.00
    rate = 0.05
    years = 10

    final = compound_interest(principal, rate, years)
    total_interest = final - Decimal(str(principal))

    print(f"Principal: ${principal:.2f}")
    print(f"Rate: {rate * 100}% annual")
    print(f"Years: {years}")
    print(f"Final amount: ${final}")
    print(f"Total interest earned: ${total_interest}")

import random

def dice_simulator(num_dice=2, rolls=1000):
    counts = {}
    for _ in range(rolls):
        total = sum(random.randint(1, 6) for _ in range(num_dice))
        counts[total] = counts.get(total, 0) + 1

    print(f"Rolling {num_dice} dice {rolls} times:")
    for total in sorted(counts.keys()):
        print(f"Sum {total:2}: {counts[total]:4} times ({counts[total] / rolls * 100:.1f}%)")

❓ Frequently Asked Questions — Numbers

Common questions about Python numbers, answered.

What is the difference between int and float?

+

int represents whole numbers (unlimited precision). float represents real numbers with a decimal point (double-precision, limited precision). Use int for counting and indexing, float for measurements and scientific calculations.

Why does 0.1 + 0.2 not equal 0.3?

+

Because 0.1 and 0.2 cannot be represented exactly in binary floating-point. The decimal 0.1 is an infinite binary fraction, just like 1/3 is an infinite decimal fraction. Use decimal.Decimal for exact decimal arithmetic.

How do I check if a number is even or odd?

+

Use the modulus operator %: n % 2 == 0 for even, n % 2 == 1 for odd. For negative numbers, n % 2 returns 0 for even and 1 for odd.

What is the difference between / and //?

+

/ is true division and always returns a float. // is floor division and returns the integer quotient, rounding down toward negative infinity.

How do I generate a random integer between 1 and 10?

+

Use random.randint(1, 10). This includes both 1 and 10. For a range with step, use random.randrange(1, 11).

What is the math module used for?

+

The math module provides mathematical functions like sqrt(), sin(), cos(), log(), exp(), and constants like pi and e. It's used for scientific and engineering calculations.

How do I format a number with thousands separators?

+

Use f-strings: f"{value:,.2f}" for 2 decimal places with thousands separators. For example, f"{1234567.89:,.2f}" gives "1,234,567.89".

What is the difference between round() and math.trunc()?

+

round() rounds to the nearest integer (banker's rounding for .5). math.trunc() truncates toward zero (removes the fractional part). For positive numbers they are the same, but for negative numbers, round(-3.7) is -4 while trunc(-3.7) is -3.
