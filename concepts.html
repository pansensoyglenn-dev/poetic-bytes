4. Factorial (loop) iteration
for loop accumulator
The factorial of a non-negative integer n (written n!) is the product of all positive integers from 1 to n. This implementation uses a for loop with range(1, n+1) to iterate through each factor. The variable fact accumulates the result by multiplying with each i. It also handles the edge case 0! correctly.

6. Prime number check math
divisibility test
A prime number is greater than 1 and has no positive divisors other than 1 and itself. This program checks primality by testing divisibility from 2 up to the square root of n (using math.isqrt). If any divisor is found, it prints "Composite" and breaks out of the loop.
import math n = int(input("Enter integer: ")) if n < 2: print("Not prime") else: for i in range(2, int(math.isqrt(n)) + 1): if n % i == 0: print("Composite") break else: print("Prime")
‚èé 17 ‚Üí prime

15. Linear search list iteration

16. Bubble sort sorting
adjacent swaps
Bubble sort repeatedly steps through a list, compares adjacent elements, and swaps them if they are in the wrong order. The pass through the list is repeated until no swaps are needed. This implementation uses two nested loops: the outer loop controls the number of passes, and the inner loop performs the comparisons and swaps.
nums = [64, 34, 25, 12, 22, 11, 90] n = len(nums) for i in range(n - 1): for j in range(n - 1 - i): if nums[j] > nums[j + 1]: nums[j], nums[j + 1] = nums[j + 1], nums[j] print("Sorted:", nums)
‚èé sorted list

21. Swap two variables tuple unpack

Swapping two variables in most languages requires a temporary variable. Python allows a more elegant way using tuple unpacking: a, b = b, a. The right‚Äëhand side creates a tuple (b, a), which is then unpacked into a and b. This operation is atomic and avoids a temporary variable.

A2. Generators and yield from
Generators produce values lazily, making them memory-efficient for large datasets. yield from delegates to another generator.
def fibonacci(): a, b = 0, 1 while True: yield a a, b = b, a + b def take(n, iterable): for i, val in enumerate(iterable): if i >= n: break yield val fib = fibonacci() print(list(take(10, fib)))
