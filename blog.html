‚ú¶ 1. DATA CLASSES ‚Äì LESS BOILERPLATE

Introduced in Python 3.7, @dataclass automatically generates __init__, __repr__, __eq__, and more. It makes class definitions concise and readable, especially for data containers.

from dataclasses import dataclass

@dataclass
class Developer:
    name: str
    language: str = "Python"
    experience_years: int = 0

dev = Developer("Dana")
print(dev)   # Developer(name='Dana', language='Python', experience_years=0)

Data classes work seamlessly with type hints and support immutable versions via frozen=True.

‚ú¶ 2. TYPE HINTS ‚Äì BEYOND BASICS

Since Python 3.5+, type hints have become ubiquitous. Modern code uses TypedDict, Literal, Union (| syntax in 3.10+), and TypeVar. Static checkers like mypy and pyright catch bugs before runtime.

from typing import Literal, TypedDict

class Config(TypedDict):
    env: Literal["dev", "prod"]
    timeout: float | int   # union via |

def set_config(cfg: Config) -> None:
    print(f"Environment: {cfg['env']}")

‚ú¶ 3. CONTEXT MANAGERS ‚Äì BEYOND WITH OPEN

Custom context managers using __enter__/__exit__ or contextlib.contextmanager simplify resource management. They‚Äôre perfect for database connections, locks, and temporary changes.

from contextlib import contextmanager

@contextmanager
def temporary_alias(obj, new_name):
    original = obj.__name__
    obj.__name__ = new_name
    try:
        yield
    finally:
        obj.__name__ = original

class Service: pass
with temporary_alias(Service, "API"):
    print(Service.__name__)   # API
print(Service.__name__)       # Service

‚ú¶ 4. DECORATORS ‚Äì POWERFUL & CLEAN

Decorators extend callables without modifying them. They‚Äôre used for logging, caching, permissions, and more. Always use functools.wraps to preserve metadata.

from functools import wraps
import time

def timer(func):
    @wraps(func)
    def wrapper(*args, **kwargs):
        start = time.perf_counter()
        result = func(*args, **kwargs)
        print(f"{func.__name__} took {time.perf_counter()-start:.4f}s")
        return result
    return wrapper

@timer
def process_data():
    return sum(i*i for i in range(10**6))

‚ú¶ 5. ASYNC/AWAIT ‚Äì CONCURRENT PYTHON

Asyncio (since 3.4/3.5) enables cooperative multitasking. Perfect for I/O-bound tasks. Modern frameworks like FastAPI and aiohttp are built on it.

import asyncio

async def fetch_data(delay, name):
    await asyncio.sleep(delay)
    return f"Data from {name}"

async def main():
    tasks = [
        fetch_data(1, "API A"),
        fetch_data(0.5, "API B")
    ]
    results = await asyncio.gather(*tasks)
    print(results)

asyncio.run(main())
