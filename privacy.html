Poetic Bytes

Privacy Policy

1. Introduction

This Privacy Policy describes how personal information is handled when you visit Poetic Bytes (https://pansensoyglenn-dev.github.io/poetic-bytes/), a personal static portfolio website showcasing Python web development work.

The site is hosted on GitHub Pages, a static hosting service provided by GitHub, Inc. As a purely static site with no backend, database, forms, comments, or user accounts, data collection is extremely limited.

2. Information We Collect

2.1 Automatically Collected Data

When you visit the site, the following information may be collected by GitHub Pages and/or your browser/network:

  • IP address
  • Browser type and version
  • Operating system
  • Referring URL (if you clicked a link from another site)
  • Timestamp and requested page/path
  • HTTP status code and bytes transferred

This is standard server access log data collected by GitHub for operational and security purposes. The site owner (Glenn Pansensoy) does not have direct access to these logs.

2.2 Information You Voluntarily Provide

None. The website does not contain contact forms, newsletter sign-ups, login systems, comment sections, or any mechanism that asks you to submit personal data.

3. Third-Party Services & Embedded Content

The site may link to or embed content from external platforms, including:

  • GitHub (repository links, gists, profile)
  • X / Twitter (if social cards or embeds are added in the future)
  • Other developer tools or CDNs (e.g., fonts, icons)

These third-party services may collect data according to their own privacy policies when you interact with their content. We do not control their data practices.

4. Cookies & Tracking Technologies

As detailed in our Cookie Policy, this site does not set any first-party cookies. No analytics, advertising, or tracking scripts (Google Analytics, Meta Pixel, etc.) are currently implemented.

Any cookies present would originate solely from third-party embedded content (if any) or browser extensions.

5. Data Retention & Security

Because we do not collect or store personal data on our servers, there is no personal data retention by the site owner.

GitHub retains server logs for a limited period in accordance with their Privacy Statement.

The site uses HTTPS (provided by GitHub Pages) to encrypt traffic between your browser and the server.

6. Your Rights

Under applicable data protection laws (e.g., GDPR, CCPA), you may have rights regarding personal data. However, since we do not collect identifiable personal information, most rights (access, rectification, erasure, etc.) do not apply directly to this site.

For questions about data processed by GitHub, please refer to GitHub’s privacy documentation.

7. Changes to This Policy

If the site begins collecting personal data in the future (e.g., through a contact form, newsletter, or analytics tool), this policy will be updated accordingly. The effective date will be revised.
