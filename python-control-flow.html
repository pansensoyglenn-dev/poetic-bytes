🔀 if, elif, else — Conditional Execution

Make decisions in your code based on boolean conditions.

📝 1.1 if Statement

The simplest conditional: execute a block only if the condition is True.

Step-by-step explanation:

  • Step 1: Define a condition, e.g., age = 18.
  • Step 2: Write if age >= 18:.
  • Step 3: Indent the block to execute when the condition is true.
python 
age = 18

if age >= 18:
    print("You are an adult.")
    print("You can vote.")

📝 1.2 if/else — Two Paths

Handle both True and False cases.

Step-by-step explanation:

  • Step 1: Check temperature > 30.
  • Step 2: If true, print hot message; else, print not hot.
python 
temperature = 25

if temperature > 30:
    print("It's hot outside.")
else:
    print("It's not too hot.")

📝 1.3 if/elif/else — Multiple Conditions

Chain multiple conditions with elif.

Step-by-step explanation:

  • Step 1: Assign score = 85.
  • Step 2: Check conditions from top to bottom.
  • Step 3: Execute the first true block.
python 
score = 85

if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
elif score >= 70:
    print("Grade: C")
elif score >= 60:
    print("Grade: D")
else:
    print("Grade: F")

🔄 for Loop — Iterating Over Sequences

Iterate over sequences like lists, tuples, strings, and ranges.

📝 2.1 Basic for Loop

Iterate over items in a collection.

Step-by-step explanation:

  • Step 1: Create a list: colors = ["red", "green", "blue"].
  • Step 2: Use for color in colors:.
  • Step 3: Print each color.
python 
colors = ["red", "green", "blue"]

for color in colors:
    print(f"Color: {color}")

📝 2.2 range() — Numeric Iteration

range() generates a sequence of numbers for iteration.

Step-by-step explanation:

  • Step 1: Use range(5) to generate 0–4.
  • Step 2: Use range(1, 6) for 1–5.
  • Step 3: Use range(0, 10, 2) for even numbers.
python 
print("range(5):")
for i in range(5):
    print(i, end=" ")
print("\n")

print("range(1, 6):")
for i in range(1, 6):
    print(i, end=" ")
print("\n")

print("range(0, 10, 2):")
for i in range(0, 10, 2):
    print(i, end=" ")

📝 2.3 enumerate() — Index and Value

Get both index and value while iterating.

Step-by-step explanation:

  • Step 1: Use enumerate() on a list.
  • Step 2: Unpack index and value in the loop.
python 
fruits = ["apple", "banana", "cherry"]

for index, fruit in enumerate(fruits):
    print(f"{index}: {fruit}")

⏳ while Loop — Repeat Until Condition

Execute a block repeatedly while a condition is true.

📝 3.1 Basic while Loop

Use when you don't know the number of iterations in advance.

Step-by-step explanation:

  • Step 1: Initialize a counter count = 0.
  • Step 2: Write while count < 5:.
  • Step 3: Print count and increment.
python 
count = 0

while count < 5:
    print(f"Count: {count}")
    count += 1

print("Loop finished!")
⚠️ Ensure the condition eventually becomes False to avoid infinite loops.

📝 3.2 User Input with while

Keep asking until valid input is received.

Step-by-step explanation:

  • Step 1: Set valid = False.
  • Step 2: While not valid, ask for input and validate.
python 
valid = False

while not valid:
    response = input("Enter a positive number: ")
    try:
        num = float(response)
        if num > 0:
            valid = True
            print(f"Thank you! You entered {num}")
        else:
            print("Number must be positive.")
    except ValueError:
        print("That's not a valid number.")

⏹️ break & continue — Loop Control

Fine-tune loop execution: exit early or skip iterations.

📝 4.1 break — Exit the Loop

Terminate the loop immediately.

Step-by-step explanation:

  • Step 1: Loop over numbers 1–10.
  • Step 2: If number is 5, break.
python 
for i in range(1, 11):
    if i == 5:
        print("Breaking at 5")
        break
    print(i)

📝 4.2 continue — Skip to Next Iteration

Skip the rest of the current iteration and move to the next.

Step-by-step explanation:

  • Step 1: Loop over numbers 0–9.
  • Step 2: If number is even, continue.
  • Step 3: Print only odd numbers.
python 
for i in range(10):
    if i % 2 == 0:
        continue
    print(i)

🚫 pass — The Null Statement

Do nothing — use as a placeholder where syntax requires a statement.

📝 5.1 Using pass

pass is often used in function stubs, class stubs, and loops that need a placeholder.

Step-by-step explanation:

  • Step 1: Define a function that does nothing yet.
  • Step 2: Use pass in a loop to skip processing.
python 
def future_function():
    pass

print("Defined a placeholder function")

for i in range(3):
    if i == 1:
        pass
    print(i)
💡 pass is not a comment — it's a valid Python statement that does nothing. It's useful for scaffolding code.

🏁 else with Loops — Execute When No break

The else block runs if the loop completes without a break.

📝 6.1 for/else and while/else

Useful for search patterns: execute else if item not found.

Step-by-step explanation:

  • Step 1: Search for a target in a list.
  • Step 2: If found, break. If not, the else runs.
python 
items = [1, 2, 3, 4, 5]
target = 3

for item in items:
    if item == target:
        print(f"Found {target}!")
        break
else:
    print(f"{target} not found.")

🧩 Nested Loops — Loops Inside Loops

Use nested loops to iterate over multi-dimensional data or generate combinations.

📝 7.1 Nested for Loops

Iterate over rows and columns of a grid.

Step-by-step explanation:

  • Step 1: Outer loop over rows.
  • Step 2: Inner loop over columns.
  • Step 3: Print a multiplication table.
python 
for i in range(1, 4):
    for j in range(1, 4):
        print(f"{i}*{j}={i*j:2}", end=" ")
    print()

🎯 match/case — Pattern Matching (Python 3.10+)

Modern and powerful alternative to long if/elif chains for structural pattern matching.

📝 8.1 Basic match/case

Match a value against multiple patterns.

Step-by-step explanation:

  • Step 1: Use match with a subject.
  • Step 2: Define case patterns.
  • Step 3: Use _ as wildcard.
python 
def describe_number(n):
    match n:
        case 0:
            return "zero"
        case 1:
            return "one"
        case 2:
            return "two"
        case _:
            return "other"

print(describe_number(0))
print(describe_number(2))
print(describe_number(5))

📝 8.2 Pattern Matching with Tuples and Guards

Match data structures and add conditions with guards.

Step-by-step explanation:

  • Step 1: Match a tuple (x, y).
  • Step 2: Use if guard for additional condition.
python 
def location(point):
    match point:
        case (0, 0):
            return "origin"
        case (0, y):
            return f"on y-axis at {y}"
        case (x, 0):
            return f"on x-axis at {x}"
        case (x, y) if x == y:
            return f"on diagonal line at ({x}, {y})"
        case _:
            return "somewhere else"

print(location((0, 0)))
print(location((0, 5)))
print(location((3, 0)))
print(location((3, 3)))
print(location((1, 2)))

❓ Frequently Asked Questions — Python Control Flow

Common questions about control flow in Python, answered.

What is the difference between break and continue?

+

break exits the loop entirely, while continue skips the rest of the current iteration and moves to the next one.

When would I use a while loop instead of a for loop?

+

Use while when the number of iterations is unknown, such as when waiting for user input or processing data until a condition is met. Use for when iterating over a known sequence.

What does the else clause do in a loop?

+

The else block runs if the loop completes without a break. It's useful for search patterns: "if not found, do something".

Is match/case available in all Python versions?

+

No, match/case was introduced in Python 3.10. For older versions, use if/elif chains or dictionaries for dispatching.

Can I use pass inside a function?

+

Yes, pass is commonly used as a placeholder in functions, classes, or loops that you haven't implemented yet. It prevents syntax errors.

How do I break out of nested loops?

+

You can use a flag variable, or wrap the loops in a function and use return. Python doesn't have a "break out of all loops" statement.

What is the difference between if and elif?

+

elif is short for "else if". It allows you to check multiple conditions sequentially. Only the first true condition's block executes.
