💡 Tips & Best Practices

Built-in modules – The runner includes the Python standard library. You can import math, random, datetime, json, csv, re, itertools, collections, and many others.

Async support – Use asyncio.run() to run async code. The runner supports async/await syntax.

Time limit – Long-running code (over 10 seconds) may be terminated to keep the page responsive.

No installation – Everything runs in your browser using Pyodide, a Python runtime compiled to WebAssembly.

PEP 8 – Write clean code following Python's style guide. Use 4 spaces for indentation, meaningful variable names, and keep lines under 79 characters.

Type Hints – Use type hints to make your code more readable and catch errors early: def greet(name: str) -> str:

Docstrings – Document your functions with docstrings: """This function does something."""

🚨 Common Python Errors & Solutions

SyntaxError: invalid syntax

Check for missing parentheses, colons, or incorrect indentation. Python is very strict about syntax.

print("Hello" # Missing closing parenthesis

NameError: name 'x' is not defined

You're trying to use a variable that hasn't been assigned a value yet.

print(x) # x was never defined

TypeError: can only concatenate str (not "int") to str

You're trying to combine different data types without converting them.

print("Age: " + 25) # Should be: print("Age: " + str(25))

IndexError: list index out of range

You're accessing an index that doesn't exist in the list.

items = [1, 2, 3]; print(items[5])

KeyError: 'key'

You're trying to access a dictionary key that doesn't exist.

d = {"name": "Alice"}; print(d["age"]) # Use d.get("age", "default")

ValueError: invalid literal for int()

You're trying to convert a non-numeric string to an integer.

int("abc") # Use try/except to handle this

ZeroDivisionError: division by zero

You're trying to divide a number by zero.

10 / 0 # Check the denominator before dividing

FileNotFoundError: [Errno 2] No such file or directory

You're trying to open a file that doesn't exist.

open("nonexistent.txt") # Use try/except to handle this

IndentationError: expected an indented block

Python uses indentation to define code blocks. Make sure you indent consistently.

def hello():\nprint("Hello") # Indent the print statement

ImportError: No module named 'xyz'

You're trying to import a module that doesn't exist or isn't installed.

import non_existent_module # Check the module name
